PUT YOUR BRAND FILES HERE — src/assets/brand/

NOT in dist/assets/brand (that folder is deleted every time you run `npm run build`).

Suggested files:
  logo.png or logo.svg     → sidebar “Fishin Leads CRM” area (bundled)
  favicon.ico              → browser tab (bundled, injected on load)
  favicon.svg              → optional second tab icon
  apple-touch-icon.png     → optional iOS home screen icon

After adding files, restart `npm run dev` if it was already running.

You can also drop copies in public/brand/ for extra fallbacks (same filenames as above).
