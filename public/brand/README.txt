Put your images here (same folder as this file).

On `npm run dev` or `npm run build`, Vite writes __manifest__.json listing your PNG/JPG/SVG/WebP/GIF/ICO files (except this README). The app uses that list so your logo and favicon work with ANY filename.

Tips:
  • Sidebar picks the best match (names with "logo", "brand", "fishin", etc. win). Favicon picks favicon.ico / favicon.svg first.
  • After adding files, save or restart dev once so the manifest refreshes (or it updates when the watcher sees changes).
  • If the tab icon doesn’t update, hard-refresh (Ctrl+Shift+R).
